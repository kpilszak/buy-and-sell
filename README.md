## General info
This project is an advertisement website.  
The main purpose is to buy and sell products.    
Application shows listing of products, owned products and enables user to edit owned products.    
	
## Technologies
Project is created with:
* Angular 14
* TypeScript
* HTML5
* Node.js
* hapi
* Firebase Auth
* MySQL
